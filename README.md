# PowerShell + cmd en menú conceptual W11
Básicamente un script .bat que añade una clave de registro. Esta clave de registro, sirve para añadir un "Abrir PowerShell como administrador" desde la ruta a la que hemos hecho click derecho, y lo mismo pero para la CMD.
